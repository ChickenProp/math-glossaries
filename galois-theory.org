- $ùîΩ_p$ :: The finite field $‚Ñ§/(p)$, where $p$ is prime.
- $ùîΩ_q$ :: A finite field of order $q = p^n$. Exists and is unique, but is not $‚Ñ§/(p)$ unless $n = 1$.

* Definitions
- Abelian :: An extension $L/K$ is abelian if it is Galois, and $\gal(L/K)$ is abelian.
- Algebraic :: $Œ± ‚àà L$ is algebraic over $K$ if there is $f ‚àà K[X]$ such that $f(Œ±) = 0$.
               \par $L/K$ is algebraic if every $Œ± ‚àà L$ is algebraic over $K$.
- Algebraically closed :: $K$ is algebraic if all algebraic extensions of $K$ are equal to $K$; equivalently if all finite extensions of $K$ are equal to $K$; equivalently if every polynomial $f ‚àà K[X]$ splits in $K$; equivalently if every nonconstant $f ‚àà K[X]$ has a root in $K$.
- Automorphism :: A $K$-automorphism $L ‚Üí L$ is an automorphism on $L$ which restricts to the identity on $K$.
- Automorphism group :: $\aut(L/K)$ is the group of all $K$-automorphisms $L ‚Üí L$.
     \par If $L^{\aut(L/K)} = K$, then $\aut(L/K)$ is also written $\gal(L/K)$.
- Constructible :: $L/K$ is constructible if there is a sequence of extensions $K_1/K, K_2/K_1, ‚Ä¶, K_n/K_{n-1}$ with $L ‚äÇ K_n$ and each $[K_{i+1} : K_i] ‚àà \{1,2\}$.
- Cyclotomic polynomial :: $Œ¶_n = \prod_Œ∂ (X - Œ∂) ‚àà ‚Ñ§[X]$ where $Œ∂$ are the primitive $n$'th roots of unity. $X^n - 1 = \prod_{d|n} Œ¶_d$.
- Fixed field :: For $H ‚â§ \aut(L/K)$, $L^H$ is the intermediate field ($K ‚â§ L^H ‚â§ L$) containing those elements of $L$ which are fixed by every element of $H$. Also written $\fix(H)$ if $L$ is clear from the context. $L^H = \{ Œ± ‚àà L: g ‚àà H ‚áí g(Œ±) = Œ± \}$.
- Frobenius automorphism :: when $K$ is a finite field of characteristic $p$, this is the map $F: K ‚Üí K: x \mapsto x^p$.
- Galois extension :: $L/K$ is /Galois/ if $L^{\aut(L/K)} = K$. That is, no $Œ± ‚àà L$ is preserved by every $K$-automorphism $L ‚Üí L$.
- Galois group :: $\gal(L/K)$ is $\aut(L/K)$, when $L/K$ is Galois.
- Normal :: $L/K$ is normal if for each $Œ± ‚àà L$, $f_Œ±^K$ splits in $L$.
            \par All Galois extensions are normal, and $L/K$ is normal iff $L$ is the splitting field for some polynomial $f ‚àà K[X]$.
- Normal closure :: A normal closure of a finite extension $L/K$ is a minimal $M/L$ such that $M/K$ is normal. A normal closure always exists, is finite, and any two normal closures are $L$-isomorphic.
- Perfect :: A field is called perfect if all of its algebraic extensions are separable. All extensions of $‚Ñö$, algebraically closed fields, and finite fields are perfect.
- Primitive :: A polynomial $f ‚àà R[X]$ is primitive if there is no irreducible $p ‚àà R$ such that $p | f$.
               \par If $g,h ‚àà R[X]$ are primitive then so is $gh$.
               \par If $g', h' ‚àà \ffrac(R)[X]$ and $f = g'h'$ is nonconstant primitive, then $‚àÉ a,b ‚àà \ffrac(R): ag', bh' ‚àà R[X]$ are primitive and $f = abg'h'$.
               \par A field extension $L/K$ is primitive if $L = K(Œ±)$ for some $Œ±$.
               \par A primitive $n$'th root of unity is one which has order $n$.
- Radical :: A finite extension $L/K$ is radical if we can write $L = K(Œ±_1, ‚Ä¶, Œ±_r)$ with integers $n_1, ‚Ä¶ n_r$ such that for each $i$, $Œ±_i^{n_i} ‚àà K(Œ±_1, ‚Ä¶, Œ±_{i-1})$.
- Separable polynomial :: An irreducible polynomial $f ‚àà K[X]$ is seperable if its derivative is nonzero; equivalently if $f$ has no repeated root in any larger field $L/K$; equivalently if $f$ splits into distinct linear factors over its splitting field.
     \par Every polynomial is separable over a field of characteristic $0$ (an extension of $\mathbb{Q}$).
- Separable extension :: $L/K$ is separable if for each $Œ± ‚àà L$, $f_Œ±^K$ is separable.
     \par An extension is Galois iff it is normal and separable. (i.e.\nbsp{}iff for each $Œ± ‚àà L$, $f_Œ±^K = c(X-Œ±_1)\cdots(X-Œ±_n)$ where the $Œ±_i$ are distinct elements of $L$.)
- Solvable :: Finite $L/K$ is solvable if it is contained in some radical extension. $f ‚àà K[X]$ is solvable if its splitting field is solvable. A group $G$ is solvable if there is a chain $G = G_0 ‚äµ G_1 ‚äµ ‚Ä¶ ‚äµ G_n = \{1\}$, such that each $G_i / G_{i+1}$ is abelian.
- Split :: $f ‚àà K[X]$ splits in $L ‚äÇ K$ if $f = c(X-Œ±_1)\cdots(X-Œ±_n)$ where $Œ±_i ‚àà L$. (i.e.\nbsp{}all roots of $f$ are in $L$.)
- Splitting field :: $L/K$ is a splitting field for $f ‚àà K[X]$ if
  - $f$ splits in $L$
  - $L = K(Œ±_1, ‚Ä¶, Œ±_n)$ where $Œ±_i$ are roots of $f$ in $L$ (i.e.\nbsp{}$L$ is not "too big").

* Theorems

** Irrecuducibility of polynomials

- Remainder theorem :: If $f ‚àà K[X]$ and $Œ± ‚àà K$, then $(X - Œ±)|f$ iff $f(Œ±) = 0$.
- Gauss' lemma :: If $f ‚àà R[X]$ is nonconstant primitive and irreducible over $R$, then it is irreducible over $\ffrac(R)$.
- Reduction :: Suppose $f ‚àà R[X]$ is primitive, $p ‚àà R$ is irreducible and $(f \mod p)$ is irreducible in $(R/(p))[X]$. Then $f$ is irreducible in $R[X]$.
- Eisenstein's criterion :: Suppose $f ‚àà R[X]$ primitive and $‚àÉp ‚àà R$ irreducible such that: $p \nmid a_n$; $p \mid a_i$ for $0 ‚â§ i < n$; $p \mid a_0^2$. Then $f$ is irreducible over $R$ and $\ffrac(R)$.

** Field extensions

- If $Œ±$ is algebraic then $K(Œ±) ‚âÖ K[X]/(f_Œ±^K)$, $Œ± \mapsto X$. If $n = \deg_K(Œ±)$ then $[K(Œ±) : K] = n$ and $1, Œ±, ‚Ä¶, Œ±^{n-1}$ is a basis of the $K$-vector space $K(Œ±)$.
- If $Œ±$ is transcendental then $K(Œ±) ‚âÖ K(X)$, $Œ± \mapsto X$.
- Finite field extensions are algebraic.
- If $K(Œ±)$ is an algebraic extension and $L/K$ is any extension, there is a bijection
  \[ \{ K\text{-homomorphisms } K(Œ±) ‚Üí L \} ‚Üí \{ \text{roots of } f_Œ±^K \text{ in } L \} \]
  mapping $œÜ \mapsto œÜ(Œ±)$. (Note, $Œ±$ need not be in $L$.)
- *Tower law* $[M : K] = [M : L][L : K]$
- If $L/K$ is a finite field extension, then every $K$-homomorphism $L ‚Üí L$ is a $K$-automorphism.

** Automorphism groups and intermediate fields

- There is a map $\{ \text{intermediate fields of } L/K \} ‚Üí \{ \text{subgroups of } \aut(L/K) \}$, given by $L' ‚Üí \aut(L/L')$.
- $L/N/M/K$ field extensions with $N/M$ finite. Then $[ \aut(L/M) : \aut(L/N) ] ‚â§ [ N : M ]$.
- Corollary: $L/K$ finite $‚áí |\aut(L/K)| ‚â§ [ L : K ]$.
- If $H ‚â§ G ‚â§ \aut(L/K)$ then $[L^H : L^G] ‚â§ [ G : H ]$.
- Special case: if $H$ is trivial we have $[L:L^G ‚â§ |G|$.
- Corollary: $[L : L^{\aut(L/K)}] ‚â§ |\aut(L/K)| ‚â§ [L:K]$.
- If $L/K$ is finite then it is Galois iff $|\aut(L/K)| = [L:K]$.
- Special case: if $L = K(Œ±)$ with $[L:K]=n$, then $L/K$ is Galois iff $f_Œ±^K$ has precisely $n$ distinct roots in $L$.
- *Fundamental theorem of Galois theory* Let $L/K$ be a finite Galois extension. There is an inclusion-reversing bijection
     \[ \{ M : L/M/K \} ‚Üî \{ H : H ‚â§ \aut(L/K) \} \]
     mapping $M \mapsto \aut(L/M)$ and $L^H \mapsfrom H$. Moreover, $L/M$ is Galois; $[L:M] = |H|$ and $[M:K] = [\gal(L/K):H]$; $M/K$ is Galois iff $H ‚ä¥ \gal(L/K)$ and these imply $\gal(M/K) = \gal(L/K)/H$.
- For $L/M/K$ with $L/K$ Galois, TFAE:
  - $\gal(L/M) ‚ä¥ \gal(L/K)$
  - For each $œÉ ‚àà \gal(L/K)$, $œÉ(M) = M$
  - $M/K$ is Galois
  - $M/K$ is normal

** Splitting fields

- Given nonzero $f ‚àà K[X]$ of degree $n$, there is a splitting field $L$ with $[L:K] ‚â§ n!$.
- *Isomorphism extension theorem* If $œÉ:K‚ÜíK'$ is an isomorphism, $f ‚àà K[X]$, $L$ is a splitting field of $f$ and $L'$ is a splitting field of $œÉ(f)$, then there is an isomorphism $L ‚Üí L'$ extending $œÉ$.
- Corollary: $f ‚àà K[X]$ has a unique splitting field up to $K$-isomorphism.
- For a finite extension L/K, TFAE:
  - $L/K$ is normal
  - $L/K$ is a splitting field for some $f ‚àà K[X]$
  - For any extension $M/L$ and $K$-homomorphism $œÉ:L‚ÜíM$, $œÉ(L) = L$.
- Any finite extension $L/K$ has a normal closure $N/L$ which is finite and unique up to $L$-isomorphism.
- Every field $K$ has an algebraic closure, unique up to $K$-isomorphism.

** Separability

- $f ‚àà K[X]$ has a repeated root $a$ iff $X - a$ divides both $f$ and $f'$.
- *Separable polynomials* If $f ‚àà K[X]$ is irreducible, TFAE:
  - $f$ has no repeated roots in any extension $L/K$
  - $f$ splits into distinct linear factors over its splitting field
  - $f' ‚â† 0$.
- If $\fchar K = 0$, every irreducible polynomial is separable.
- If $\fchar K = p > 0$, $f$ irreducible, then $f$ is separable iff there is $g ‚àà K[X]$ such that $f = g(X^p)$.
- A finite extension $L/K$ is Galois iff it is normal and separable; iff it is the splitting field of some $f ‚àà K[X]$ such that all irreducible factors of $f$ are separable.
- *Primitive extension theorem* finite $L/K$ is primitive iff there exist only finitely many intermediate fields.
- Corollary: Finite degree separable extensions are primitive.

** Finite fields

- A finite subgroup of $K^*$ is cyclic.
- If $L/K$ are both finite fields, the extension is primitive.
- $|K| = p^n$ iff $K$ is a splitting field of $X^n - X ‚àà ùîΩ_p[X]$.
- Corollary: there is a unique finite field of order $p^n$.
- If $|K| = p^n$ then $K/ùîΩ_p$ is Galois, and $\gal(K/ùîΩ_p)$ is cyclic of order $n$, generated by the Frobenius automorphism $F$.
- Corollary: the subfields of $ùîΩ_{p^n}$ are $ùîΩ_{p^m}$ for each $m|n$.
- Corollary: if $L$ is finite then any $L/K$ is Galois, with $\gal(L/K)$ generated by a power of $F$.

** Cyclotomic fields

- Let $L/K$ be any field extension and $Œ∂ ‚àà L$ a primitive $n$'th root of $1$. Then $K(Œ∂)/K$ is Galois, and there is an injective homomorphism $\gal(K(Œ∂)/K) ‚Üí (‚Ñ§/n‚Ñ§)^* : œÉ \mapsto (k \text{ such that } œÉ(Œ∂) = Œ∂^k)$.
- If $K = ‚Ñö$ above, then the homomorphism is an isomorphism, and $Œ¶_n$ is irreducible.
- Corollary: any subfield of $‚Ñö(Œ∂)$, where $Œ∂$ is any root of unity, is abelian over $‚Ñö$.
- Any abelian extension of $‚Ñö$ is a subfield of some $‚Ñö(Œ∂)$. (Proof nonexaminable.)
- The regular $n$-gon can be constructed from two points in the plane iff $n$ is of the form $n = 2^m \prod_{i=1}^k p_i$, where the $p_i$ are distinct Fermat primes (primes of the form $2^j + 1$).

** Solvability in radicals

- If $x$ can be written in terms of $√ó$, $√∑$, $+$, $-$ and $\sqrt[n]{}$ ($n ‚àà ‚Ñï$) applied to elements of $K$, then $K(x)/K$ is solvable.
- If $L/K$ is radical and $N$ is the normal closure, then $N/K$ is radical.
- A finite extension $L/K$ is radical iff its normal closure $N/K$ is radical.
- Irreducible $f ‚àà K[X]$ is solvable iff there is a solvable extension $L/K$ with some $x ‚àà L$ such that $f(x) = 0$.
- If $Œ∂ ‚àà K$ is a primitive $n$'th root of unity, and $Œ±^n ‚àà K$, then $K(Œ±)/K$ is Galois with $\gal(K(Œ±)/K)$ cyclic of order dividing $n$.
- If $G$ is a solvable group and $H ‚â§ G$, then $H$ is solvable; if $H ‚ä¥ G$ then $G/H$ is solvable also.
- Let $\fchar K = 0$, $f ‚àà K[X]$ be solvable, and $L$ be the splitting field of $f$. Then $\gal(L/K)$ is solvable.
- If $H ‚ä¥ G$ and $G/H$ is abelian, then $[G,G] ‚â§ H$.
- $[A_5, A_5] = A_5$; as a corollary, $A_5$ is not solvable.
- If $p$ is prime, $H ‚â§ S_p$, and $H$ conains a $p$-cycle and a $2$-cycle, then $H = S_p$.
- If $f ‚àà ‚Ñö[X]$ is an irreducible quintic with three real and two complex roots, then $f$ is not solvable.
- Corollary: there is no general formula for solving quintic polynomials in radicals.

** Calculating Galois groups

In this section we let $f ‚àà K[X]$ of degree $n$, and $L$ be it's splitting field, with $f$ having no repeated roots in $L$. We write $\gal(f)$ for $\gal(L/K)$, which can be thought of as a subgroup of $S_n$, as each $œÉ ‚àà \gal(f)$ is defined by a permutation of the roots of $f$.

- Any polynomial expression in $Œ±_i$ which is symmetric (invariant under permutation of the $Œ±_i$) is a polynomial in the $a_i$ (so e.g.\nbsp{}$D$ is but $Œî$ isn't).
- Let $œÉ ‚àà \gal(f)$. Then $œÉ(Œî) = ¬±Œî$, $+$ if $œÉ$ is an even permutation and $-$ if $œÉ$ is an odd permutation of roots.
- Corollary: $\gal(f) ‚â§ A_n$ iff $Œî ‚àà K$.
- Corollary: If $f$ is irreducible and $n = 3$, then $\gal(f) = A_3$ if $Œî ‚àà K$, $S_3$ otherwise.
